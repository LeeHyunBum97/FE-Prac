git hub의 기본 branch 이름 main과 git 의 기본 branch 이름 master가 달라서 발생하는 문제를 해결해서 main에 업로드 하기

1. 프로젝트 업로드
    git hub 사이트에서 레퍼지토리를 생성하고 URL을 복사
    readme.md 파일을 생성한 경우와 그렇지 않은 경우 중간 작업이 조금 상이함.

2. 현재 프로젝트를 로컬 git과 연결
    - 로컬 git과 연결 git init

3. 변경된 내용을 로컬 git에 반영
    - git add . : u가 붙은 파일을 선택하고 플러스 버튼 눌러도 된다 -> 스테이징
    - git commit -m 메시지
